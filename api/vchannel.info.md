# vchannel.info

返回指定聊天会话的完整信息。

## 请求方式

```
GET {base_url}/vchannel.info
```

## 请求参数

**需要登录**

| 参数名称 | 参数类型 | 参数是否必须？ | 说明 | 样例 |
|:--------:|:--------:|:--------------:|------|------|
| `vchannel_id` | `string` | 是 | 聊天会话 id |  |
