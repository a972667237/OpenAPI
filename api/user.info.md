# user.info

返回团队内指定用户完整信息。

## 请求方式

```
GET {base_url}/user.info
```

## 请求参数

**需要登录**

| 参数名称 | 参数类型 | 参数是否必须？ | 说明 | 样例 |
|:--------:|:--------:|:--------------:|------|------|
| `user_id` | `string` | 是 | 指定用户 id，如 \"=bw52O\" |  |
